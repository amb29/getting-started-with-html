BASICS OF THE WEB

There are at least five important subjetcs to be studied as basic elements on this topic: the web itself, the code language HTML, the repository URL,
the protocol HTTP and proper web aplications. First of all the web - or world wide web.

The World Wide Web

The World Wide Web is a collection of HTML documents, which stands for Hypertext Markup Language. The documents are all linked to one ano-
ther throught Hyper Links. All kinds of file can exist on the web, music, video, images - HTML documents can refer to anything.
There are four main actors in the web: users, computers + browsers, the internet itself and servers. The user put up an internet adress on his browser
and it makes a request via the internet to the servers, using the HTTP protocol. The servers respond with files displaced by the browser. Servers are com-
puters which host files, and the HTTP is the main protocol used on internet communications. Web documents, as said before, are written in a language
called HTML, the next topic.

HTML - Hyper Text Markup Language

HTML or Hyper Text Markup Language is the language used to write documents placed on the web. It is made of four components: text content, or
the text that can be seen; Markup, or what the text looks like; References to other documents; and Links to other pages. Text content is plain text, while
markup is used to mark texts up.

HTML Markup

Texts are marked up using tags. Usually an element is made of an openinge tab, a closing tab and contents, for example:

Ex: < >contents< > ----- This is called an element

In an element the contents inside the opening and closing tags are marked up with the effects of that tag. There are several kinds of tags, which one with
its own markup effect on text, like the b (bold) tag, em (emphasis - italics), Anchor tags (a), used to make links, image tags(img), among others. There are
also tags with attributes (tag attr = "value")", which have values. It is important to say that image tags are void tags, that don´t need a closing tab.

Whitespace

All that is put in HTML turn in a single space. If a user writes a text in multiple lines, it appears in a single line, in whitespace. It is necessary to force multi-
ple lines using tags. The break tag (br) is the first way. It is a void tag to be used after any line the users wants a break. The other way is with paragrarh tag
(p), which is not a void tag. The main difference between those two tags is that break is an inline tag, while p is a block one.

Inline and Block elements

Block and inline tags have different behaviour. Inline elements fit within a line, somentimes ending it, like break tag. Block tags make and an invisible box
around the text, that can have height and width. Examples of inline elements are: a, span, br, image strong. Some examples of block elements: div, p, form.

Container elements

Containers are elements that contain stuff inside them. There are two container tags: span and div. Span is an inline element, while div is a block one. They both
can use CSS, which is a language to give styles to documents.
